# Настройка директории браузера

## Важно: Используйте отдельную директорию!

**НЕ используйте стандартную директорию Chrome:**
```
C:\Users\YourName\AppData\Local\Google\Chrome\User Data
```

### Почему?

1. **Конфликт с запущенным Chrome:**
   - Если Chrome запущен, он блокирует доступ к своей директории
   - Агент не сможет запустить браузер

2. **Потеря данных:**
   - Может привести к повреждению профиля Chrome
   - Потеря закладок, истории, паролей

3. **Безопасность:**
   - Агент должен работать изолированно
   - Отдельная директория защищает ваш основной профиль Chrome

## Рекомендуемая настройка

### Вариант 1: Использовать директорию по умолчанию (рекомендуется)

В `.env` файле **НЕ указывайте** `BROWSER_USER_DATA_DIR` или укажите:
```env
BROWSER_USER_DATA_DIR=./browser_data
```

Агент автоматически создаст директорию `browser_data` в корне проекта.

### Вариант 2: Указать свою директорию

Если хотите использовать другую директорию:
```env
BROWSER_USER_DATA_DIR=C:\Users\YourName\ai_agent_browser
```

**Важно:** Укажите путь к **новой пустой директории**, а не к существующей директории Chrome!

## Если всё же нужно использовать стандартную директорию

⚠️ **НЕ рекомендуется**, но если необходимо:

1. **Полностью закройте Chrome:**
   - Закройте все окна Chrome
   - Проверьте диспетчер задач - не должно быть процессов `chrome.exe`
   - Подождите 5-10 секунд

2. **Укажите путь в .env:**
   ```env
   BROWSER_USER_DATA_DIR=C:\Users\YourName\AppData\Local\Google\Chrome\User Data
   ```

3. **Запустите агента**

4. **После работы агента:**
   - Закройте агента
   - Подождите несколько секунд
   - Только потом запускайте Chrome

## Проверка директории

Проверить, какая директория используется:
```powershell
# При запуске агент покажет:
📁 Директория браузера: C:\...\browser_data
```

## Очистка директории

Если нужно начать с чистого профиля:
```powershell
# Windows
Remove-Item -Recurse -Force browser_data
```

При следующем запуске агент создаст директорию заново.

## Преимущества отдельной директории

✅ Изоляция от основного Chrome  
✅ Нет конфликтов с запущенным Chrome  
✅ Можно безопасно удалить без потери данных Chrome  
✅ Агент работает независимо  
✅ Легче отлаживать проблемы
