# Руководство по использованию

## Быстрый старт

1. **Установите зависимости:**
```bash
go mod download
```

2. **Создайте файл `.env` в корне проекта:**
```bash
# Скопируйте шаблон
cp .env.example .env

# Отредактируйте .env и добавьте ваш реальный API ключ
# НИКОГДА не коммитьте .env файл!
```

Пример содержимого `.env`:
```env
OPENAI_API_KEY=sk-your-real-api-key-here
OPENAI_MODEL=gpt-4-turbo-preview
BROWSER_USER_DATA_DIR=./browser_data
ENABLE_SECURITY_LAYER=true
START_URL=https://www.google.com
```

**⚠️ БЕЗОПАСНОСТЬ:**
- Файл `.env` автоматически игнорируется git
- `browser_data/` содержит cookies и сессии - тоже игнорируется
- Используйте `.env.example` как шаблон, но не храните реальные ключи в репозитории

3. **Запустите агента:**
```bash
go run main.go
```

4. **Введите задачу в терминале:**
```
Прочитай последние 10 писем в яндекс почте и удали спам
```

## Примеры задач

### Удаление спама
```
Прочитай последние 10 писем в яндекс почте и удали спам
```

Агент:
- Откроет Яндекс.Почту
- Войдет (если нужно, вы можете войти вручную - сессия сохранится)
- Прочитает последние письма
- Определит спам
- Удалит спам-письма
- Предоставит отчет

### Заказ еды
```
Закажи мне BBQ-бургер и картошку фри из того места, откуда я заказывал на прошлой неделе
```

Агент:
- Найдет сайт доставки
- Найдет ресторан с историей заказов
- Добавит нужные позиции в корзину
- Пройдет до checkout (остановится перед оплатой для подтверждения)

### Поиск вакансий
```
Найди 3 подходящие вакансии AI-инженера на hh.ru и откликнись на них с сопроводительным письмом
```

Агент:
- Откроет hh.ru
- Изучит ваш профиль
- Найдет релевантные вакансии
- Откликнется с персонализированным сопроводительным письмом

## Особенности

### Persistent Sessions
Браузер сохраняет сессии в `./browser_data`. Вы можете:
- Войти в сервисы вручную один раз
- Агент продолжит работу с сохраненной сессией

### Security Layer
При деструктивных действиях (удаление, оплата) агент запросит подтверждение:
```
⚠️  ВНИМАНИЕ: Деструктивное действие обнаружено!
Действие: delete
Описание: Удаление письма
Продолжить? (yes/no):
```

### Автономная работа
Агент:
- Самостоятельно определяет последовательность действий
- Адаптируется при ошибках
- Не использует заготовленные планы
- Анализирует страницу в реальном времени

## Устранение неполадок

### Браузер не открывается
- Убедитесь, что Chrome/Chromium установлен
- Проверьте права доступа к `./browser_data`

### Ошибки API
- Проверьте правильность `OPENAI_API_KEY`
- Убедитесь, что у вас есть доступ к выбранной модели

### Агент застрял
- Нажмите Ctrl+C для остановки
- Проверьте логи в терминале
- Увеличьте `maxIterations` в `agent/agent.go` если нужно
