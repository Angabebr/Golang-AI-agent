# Руководство по командам агента

## Как давать команды

После запуска агента (`.\Golang-AI-agent.exe` или `go run main.go`) вы увидите приглашение:

```
> 
```

Просто введите вашу задачу текстом и нажмите Enter. Агент будет выполнять её автономно.

## Служебные команды

### help / помощь / справка
Показать справку по использованию агента

```
> help
```

### exit / quit / выход
Завершить работу агента

```
> exit
```

## Как работает агент

1. **Вы вводите команду** - описание задачи на естественном языке
2. **Агент анализирует** - понимает, что нужно сделать
3. **Агент действует** - выполняет задачу шаг за шагом в браузере
4. **Вы наблюдаете** - видите все действия агента в реальном времени
5. **Агент отчитывается** - сообщает о результате выполнения

## Что видит агент

Агент видит:
- ✅ Текущий URL страницы
- ✅ Заголовки (h1, h2, h3)
- ✅ Все ссылки на странице
- ✅ Все кнопки и их текст
- ✅ Все поля ввода и их placeholder'ы
- ✅ Текстовое содержимое страницы

Агент НЕ видит:
- ❌ Заранее подготовленные селекторы
- ❌ Предопределенные пути на сайтах
- ❌ Подсказки о структуре сайтов

## Безопасность

При деструктивных действиях (удаление, оплата, отправка данных) агент:
1. Определит, что действие опасное
2. Покажет предупреждение
3. Запросит ваше подтверждение

Пример:
```
⚠️  ВНИМАНИЕ: Деструктивное действие обнаружено!
Действие: delete
Описание: Удаление письма
Продолжить? (yes/no):
```

## Советы по использованию

### ✅ Хорошие команды:
- Конкретные и понятные
- С указанием количества (если нужно)
- С указанием сайта (если нужно)
- На русском или английском языке

### ❌ Плохие команды:
- Слишком общие ("сделай что-нибудь")
- Без контекста ("удали")
- С техническими деталями ("нажми на кнопку с id='submit'")

## Многошаговые задачи

Агент может выполнять сложные многошаговые задачи:

```
Зайди на сайт доставки еды, найди ресторан где я заказывал на прошлой неделе,
добавь в корзину BBQ-бургер и картошку фри, перейди к оформлению заказа,
но остановись перед оплатой и покажи мне что в корзине
```

Агент сам разобьет задачу на шаги и выполнит их последовательно.

## Ограничения

- Максимальное время выполнения задачи: 15 минут
- Максимальное количество итераций: 50
- Агент работает только с тем, что видит на странице
- Требуется подключение к интернету
- Требуется валидный OpenAI API ключ

## Получение помощи

Если что-то не работает:
1. Проверьте `.env` файл с API ключом
2. Убедитесь, что Chrome установлен
3. Посмотрите сообщения об ошибках в терминале
4. Проверьте `TROUBLESHOOTING.md`
